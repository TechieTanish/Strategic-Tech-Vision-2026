- Object Oriented
- Secure
- Robust

### 3. Types of Classes in Java
I learned about different types of classes:
- Normal Class
- Abstract Class
- Final Class
- Inner Class

---

### 4. Access Specifiers
Access specifiers control visibility of variables and methods:
- public
- private
- protected
- default

---

### 5. Compiler and Interpreter
I understood the difference:
- Java source code is first **compiled** by `javac`
- Bytecode is then **interpreted** by JVM
- This makes Java platform independent

I understood the basic structure of Java code:
- class keyword
- main method
- System.out.println()

Example:
```java

public class HelloWorld {

        System.out.println("Hello Java");
    }
}
